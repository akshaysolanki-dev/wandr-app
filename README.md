# Wandr 🌍

A smart local discovery web app to explore cafes, parks, and places around you.

## 🚀 Features
- Clean modern UI
- Search functionality (UI ready)
- Mood & budget filters
- Responsive layout

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Goal
To build a useful real-world web app while learning frontend development.
