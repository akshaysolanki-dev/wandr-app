# Wandr 🌍

**Wandr** is a smart local discovery web app designed to help users explore nearby cafes, parks, and hidden gems based on their mood and preferences.

---

## ✨ Highlights

* 🔍 Smart place discovery interface
* 🎯 Mood & budget-based filtering system
* 📱 Fully responsive modern UI
* ⚡ Smooth and clean grid-based layout
* 🧠 Built with real-world usability in mind

---

## 🚀 Features

* Search functionality (UI ready)
* Dynamic filter system (Mood + Budget)
* Responsive card grid layout
* Clean and minimal user interface
* Scalable structure for future backend integration

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout (Grid/Flexbox)
* **JavaScript (Vanilla)** – Interactivity
